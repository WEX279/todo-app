# To do app

En esta aplicación de navegador podrás hacre uso de un administrador de tareas en el que podrá anotar tus responsabilidades y eliminarlas una vez las hayas cumplido.

---

# Proceso de instalación:

## Descargar repositorio

Abre la consola de tu ordenador o bien por el comando Win + R, abriendo con click derecho en el escritorio o buscando "Terminal" en la barra de navegación. A continuación introduce este comando y pulsa Enter:

```bash
https://github.com/WEX279/todo-app
```
Una vez copiado, tendrás total acceso a la información

---

## Acseso la app

Desde el explorador de rchivos puedes abrir el documento con tu navegador de confianza

---
